# Lottie Animations

Store your Lottie JSON animation files in this directory.

## Structure

```
public/lottie/
├── loading/
│   ├── spinner.json
│   └── dots.json
├── success/
│   ├── checkmark.json
│   └── celebration.json
├── error/
│   └── error-icon.json
└── illustrations/
    └── empty-state.json
```

## Best Practices

1. **Naming Convention**: Use kebab-case for file names
2. **Organization**: Group animations by category or feature
3. **Optimization**: Ensure animations are optimized for web
4. **Size**: Keep files under 500KB for optimal performance
5. **Testing**: Test animations on both desktop and mobile devices

## Finding Lottie Animations

- [LottieFiles](https://lottiefiles.com/)
- [IconScout Lottie](https://iconscout.com/lottie-animations)
- [Lottie Lab](https://lottielab.com/)

## Creating Custom Animations

Use Adobe After Effects with the Bodymovin plugin or tools like:
- [LottieFiles Creator](https://lottiefiles.com/creator)
- [Haiku Animator](https://www.haikuforteams.com/)
- [Jitter](https://jitter.video/)
